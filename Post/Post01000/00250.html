문제 :

3 つの整数 A, B, C が与えられる．A, B, C はそれぞれ 1 または 2 である．1 と 2 のうち，どちらが多くあるか．
세 개의 정수 A, B, C가 주어집니다. A, B, C는 1 혹은 2인데, 1이 많습니까? 2가 많습니까?

입력 :

入力は以下の形式で標準入力から与えられる．
입력은 A B C 로 주어집니다. (공백 1개)

출력 :

1 と 2 のうち，個数が多い方を出力せよ．
1과 2중에 무엇이 더 많은지 출력해주세요.


A, B, C는 각각 1 아니면 2입니다.
1이 3개, 2가 0개면 총 합은 3입니다.
1이 2개, 2가 1개면 총 합은 4입니다.
1이 1개, 2가 2개면 총 합은 5입니다.
1이 0개, 2가 3개면 총 합은 6입니다.
즉, 총 합이 5 이상이면 2가 많은 것이고, 총 합이 4 이하이면 1이 많은 것입니다.
A, B, C = map(int, input().split())으로 각각 정수로 입력받아올 수 있습니다.

사실 A + B + C의 값만 필요한 것이니 map 상태를 list() 함수로 감싸서 [A, B, C]인 리스트 상태로 만들고, sum 함수로 합쳐버려도 됩니다.
