문제 :

(생략) 여기 에 있는 스도쿠 문제를 푼다.

입력 :

-

출력 :

예제 출력처럼 출력형식을 맞춰서 출력한다.

다음과 같은 스도쿠를 풀어야 합니다.
0 2 4 0
1 0 0 3
4 0 0 2
0 1 3 0
A4 용지도 필요없고 간단히 포스트잇에 끄적거리면서 답을 찾아줍니다.
푸는 법의 시작을 간단하게 알려드리자면,
0 2 4 Y
1 0 X 3
4 0 0 2
0 1 3 0
X에 1이 들어올 수 없으므로 (X 왼쪽에 1이 있음), 무조건 Y로 1이 들어가야 합니다. 그에따라 자연스럽게 X는 2가 됩니다. (우상단 박스에 4, 3, 1이 들어와서 남은건 2뿐)

이와 비슷한 실버급의 3*3 스도쿠 문제로 20095, 20096번이 있고, 4*4 스도쿠로 20099번이 있습니다. (다른거는 8*8, 10*10 같은 거대 스도쿠입니다. 10*10이면 칸만 1만개가 있는 셈입니다.)

이 문제와 비슷한 Text 언어로 풀 수 있는 문제들입니다.
다른 문제도 같이 풀어보시는걸 추천드립니다.
