삼각함수의 덧셈정리

삼각함수의 덧셈정리는 다음과 같습니다.

$$ \sin(x + y) = \sin x \cos y + \cos x \sin y \\ \cos(x + y) = \cos x \cos y - \sin x \sin y $$
$\sin(x)$는 사인함수 이고, $\cos(x)$는 코사인함수 입니다.
삼각함수의 덧셈 정리 증명

$$ f(x) = \sin(x + c) - \sin x \cos c - \cos x \sin c \\ g(x) = \cos(x + c) - \cos x \cos c + \sin x \sin c $$
$x$에 대해 미분하면 ... 후에 $f^2 + g^2 = 0$ 보이기, 그 식을 만족하면 $f = g = 0$을 만족하는 것도 보이기

탄젠트함수 $\tan x$에도 이와 비슷한 덧셈정리가 있습니다.
$$ \tan(x + y) = \frac{\tan x + \tan y}{1 - \tan x \tan y} $$
