문제 :

두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오.

입력 :

첫째 줄에 A, 둘째 줄에 B가 주어진다. (0 < A, B < 10)

출력 :

첫째 줄에 A+B를 출력한다.

한 줄에 하나씩 입력이 들어오므로, A = int(input()) 을 하고, B = int(input()) 을 해서 입력을 받을 수 있습니다.
이후에는 백준 01000번 - A+B 처럼, print(A + B)로 A와 B의 합을 출력할 수 있습니다.

변수 선언하기도 귀찮으시다면, 그냥 print(int(input()) + int(input()))으로 한 줄로 풀 수도 있습니다.
