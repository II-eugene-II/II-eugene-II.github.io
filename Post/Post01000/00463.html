삼차방정식의 근의 공식

$x$에 대한 삼차방정식 $a x^{3} + b x^{2} + c x + d = 0$의 근을 구하는 공식은 다음과 같습니다.

$$ x_{0} = \omega^{0} \sqrt[3]{- {q \over 2} + \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} + \omega^{0} \sqrt[3]{- {q \over 2} - \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} - \frac{b}{3a} \\ x_{1} = \omega^{1} \sqrt[3]{- {q \over 2} + \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} + \omega^{2} \sqrt[3]{- {q \over 2} - \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} - \frac{b}{3a} \\ x_{2} = \omega^{2} \sqrt[3]{- {q \over 2} + \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} + \omega^{1} \sqrt[3]{- {q \over 2} - \sqrt{\left({q \over 2}\right)^2 + \left({p \over 3}\right)^3}} - \frac{b}{3a} $$
$\omega = \frac{-1 + i \sqrt{3}}{2}$이고, $i$는 허수 단위 $i$ 입니다.
$p$는 $\left(\frac{c}{a} - \frac{1}{3} {\left( \frac{b}{a} \right)}^{2} \right)$, $q$는 $\left(\frac{d}{a} - \frac{1}{3}\frac{bc}{a^{2}} + \frac{2}{27} {\left( \frac{b}{a} \right)}^{3} \right)$ 입니다.
편의상 $P = \dfrac{p}{3}$, $Q = -\dfrac{q}{2}$라 하면 다음과 깉이 쓸 수 있습니다.
$$ x_{0} = \omega^{0} \sqrt[3]{Q + \sqrt{Q^2 + P^3}} + \omega^{0} \sqrt[3]{Q - \sqrt{Q^2 + P^3}} - \frac{b}{3a} \\ x_{1} = \omega^{1} \sqrt[3]{Q + \sqrt{Q^2 + P^3}} + \omega^{2} \sqrt[3]{Q - \sqrt{Q^2 + P^3}} - \frac{b}{3a} \\ x_{2} = \omega^{2} \sqrt[3]{Q + \sqrt{Q^2 + P^3}} + \omega^{1} \sqrt[3]{Q - \sqrt{Q^2 + P^3}} - \frac{b}{3a} $$
제곱근과 세제곱근의 방향은 $z = r e^{i \theta}$ ($-\pi \leq \theta < \pi$)일 때, $\sqrt{z} = r^{\frac{1}{2}} e^{i \theta/2}$, $\sqrt[3]{z} = r^{\frac{1}{3}} e^{i \theta/3}$로 계산해주면 됩니다.
삼차방정식의 근의 공식 증명

과정 1. $a x^{3} + b x^{2} + c x + d = 0$를 $x^{3} + B x^{2} + C x + D = 0$로 바꿀 수 있음 보이기

과정 1 증명

$a x^{3} + b x^{2} + c x + d = 0$의 양변을 $a$로 나누어주면 $x^{3} + \frac{b}{a} x^{2} + \frac{c}{a} x + \frac{d}{a} = 0$이라 할 수 있다.
$B = \frac{b}{a}$, $C = \frac{c}{a}$, $D = \frac{d}{a}$라 할 수 있다.

과정 2. $x^{3} + B x^{2} + C x + D = 0$를 $t^{3} + P t + Q$로 바꿀 수 있음 보이기
과정 2 증명

$x = \left( t - \frac{B}{3} \right)$대입
