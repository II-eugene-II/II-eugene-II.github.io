다중계승

다중계승 (Multifactorials) 은 자연수 $m$, $-m$보다 정수 범위($0$, $1$, $2$, $3$, $\cdots$) $n$에 대해 다음과 같이 정의합니다.

$$ n!_{(m)} = \begin{cases} n \times (n - m)!! & \mbox{if } n > 0 \\ 1 & \mbox{if } -m < n \leq 0 \end{cases} $$
팩토리얼 의 정말 특수한 확장으로 볼 수 있습니다.
$n$부터 $m$씩 빼가면서 곱한 수 입니다.
$m = 1$이면 그냥 팩토리얼이고, $m = 2$이면 이중계승 이라 할 수 있습니다.
