유클리드의 보조정리

유클리드의 보조정리 (Euclid's Lemma) 는 다음과 같습니다.

소수 $p$에 대하여 $ab \equiv 0 \pmod{p}$이면 $a \equiv 0 \pmod{p}$이거나 $b \equiv 0 \pmod{p}$이다.
$a \pmod{p}$는 모듈러 연산 (나머지 연산) 입니다. 즉, $p$가 $ab$의 약수 ($p \mid ab$)라는 뜻입니다.
유클리드의 보조정리 증명

증명의 스케치
베주 항등식을 통해 모순 발견 (둘 다 아니면 말도 안되는...)
$p$가 $a$의 약수 도 아니라고 가정
$\gcd(a, p) = g$라고 하면, $g \mid p$이므로 소수의 정의에서 $g = 1$ 혹은 $g = p$이나, $g = p$이면 $p$가 $a$의 약수여야 하고 이는 약수가 아니라는데에 모순, 따라서 $g = 1$
베주항등식 에서 $m a + n p = 1$인 정수 $m$, $n$이 존재한다. 양변에 $b$를 곱하면 $m a b + n b p = b$이고, $p \mid ab$에서 어떤 정수 $k$에 대해 $kp = ab$ 성립, $m a b + n b p = b$에 대입하면 $m k p + n b p = b$, 따라서 $b$가 $p$의 배수가 되어버림
같은 방법으로, $p$가 $b$의 약수가 아니라고 가정하면, $p$는 $a$의 약수가 되어버림
