램프 함수

램프 함수 (Ramp function) $R(x)$는 다음과 같이 정의합니다.

$$ R(x) = \begin{cases} 0 & x \leq 0 \\ x & x > 0 \end{cases} $$
단위 계단 함수 와는 다르게 $x = 0$일 때의 정의는 어차피 $0$이기 때문에 그 부분에는 크게 신경을 쓸 필요가 없습니다.
아이버슨 괄호 를 이용하면 다음과 같은 정의가 가능합니다.
$$ R(x) = x [ x > 0 ] $$
어차피 $x$를 곱하기 때문에 $R(x) = x [ x \geq 0 ]$으로 해도 똑같은 표기 입니다.
딥러닝 분야 등에서는 ReLU 함수로 부르기도 하고, 학습용으로 로지스틱 함수 또한 사용됩니다.
