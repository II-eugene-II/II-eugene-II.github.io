문제 :

Print a table that describes the current count of all your livestock.

-

출력 :

Print the table below as shown. The character “-”, is a dash not an underscore.
(축약 & 번역) - 아래 표를 예제 출력과 같이 출력합니다. "-"는 밑줄이 아닌 대시입니다.

'나는 코딩 실력이 좋지 않아서 브론즈 5 문제를 풀어보고 있다' 같은 경우에는 이 문제를 풀지 않고 넘어가는걸 권해드리고, '그냥 브론즈 5 문제를 싹 다 풀어보고 싶다' 하는 분에게 권해드리는 문제입니다.
