백준 티어 Platinum IV 달성

백준 티어 플레4 달성을 했습니다.
아직도 실제 실력은 플레4는 커녕 골드 될락말락 한 수준인 듯 하지만...하여간 달성을 했습니다.
앞으로 남은 티어는 14개 (플레에 남은거 3개, 다이아 5개, ...루비 5개?, ......마스터??) 가 있는데, 여기까지 오니 슬슬 다음 티어로 가는게 굉장히 어렵다는게 절실히 느껴집니다.
아마도 정말 많이 가도 다이아 어디선가 멈출 것 같은 그런 느낌이 듭니다.
티어가 오른다고 실력이 늘어난다는 생각은 딱히 가지고 있지 않은데 그냥...재미있으니까 / 기왕이면 높은게... 라는 생각으로 꾸준히 문제 풀고 점수도 올려주고 있습니다.
근래에는 브론즈 문제를 왕창 풀어주었는데 (문제에 오류가 있다고 뜨는 6841번을 빼고 브론즈 4를 싹 다 풀었습니다.) 그래도 이 쯤 되니 브론즈 문제에는 겁먹지 않게는 된 듯 합니다.
기존에는 수학 분야에 편식이 좀 심한 듯 하여, 그래프 관련 문제도 손을 대보고, 자료구조 문제도 손을 대보고, 그리디 문제도 손을 대보고, 기하학...은 수학이긴 하지만 하여간 손을 가볍게 대보았습니다.
플레5 달성 당시보단 조금 더 나아진 듯 싶지만 아직 부족하다고 생각되는 점 / 느낀 점을 하나 둘 적어보면서 앞으로의 방향을 잡아보고자 합니다.

1. 이게 수학과인지 컴공과인지 2

위에서 그래프 문제를 풀었다고 했으나 사실상 그래프를 가장한 수학문제입니다.
물론 문제 풀 때 수학이 중요한걸 점점 더 느끼고는 있지만 그래도 DFS니 BFS니 하는 그런 것들도 풀긴 풀어야겠다...라는 생각이 계속 듭니다.

2. 그래도 이제는 Class 2+

플레5 당시에 Class 2++이든 Class 3이든 얻어야겠다고 생각했는데 진짜 Class 2에 있는 문제를 모두 푸는건 상당히 어려운 일이고 에센셜 문제들은 풀어주어서 2+를 얻었습니다.
스택만 조금 건드릴 줄 알았는데, 이젠 큐니 덱이니 하는 것들도 조금은 건드릴 수 있게 되었습니다.

3. 파이썬이 너무 편하다 2

예전에 파이썬이 정말 "느린가"? 라는 주제의 글을 보았는데, 대강 요지는 1. 진짜 빠른거로 코딩하려면 어셈블리로 하던가 2. 대부분은 어셈블리 코드 작성하는 시간이 파이썬 코드 작성하는 시간 + 실행하는 시간보다 길 것 이었습니다.
이진탐색도 계속 건드릴때마다 탐색할 것이 무조건 있는 경우 / 없는 경우 / mid를 리턴해야하는지 / mid - 1인지 mid + 1인지 맨날 헷갈리면서 만드는데, 이진탐색도 import해서 그냥 쉽게 할 수 있다는 거에 당황했습니다.
언젠가는 그래도 C++도 같이 병행하긴 할 것 같은데 적응할 수 있을지 모르겠습니다.
(아마도 정수 오버플로우 자체가 어색해서 여차하면 시작하자마자 BIGint 같은 클래스부터 만들고 시작할 것 같습니다.)
플레5 시점에서는 풀었던 문제 티어 Top100에서 가장 높은게 다이아4, 낮은게 실버3(!)이었는데, 이제는 가장 낮은게 골드4가 되었습니다.
골드4 이상에서만 100문제를 풀었다고 하니 그래도 아무것도 안한건 아닌 수준이구나 싶습니다.
플레5 시점에서 푼 문제는 이랬었는데, 플레3 시점에서는 얼마나 달라졌는지 확인해보겠습니다.
