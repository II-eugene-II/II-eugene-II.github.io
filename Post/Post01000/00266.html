문제 :

연세대학교의 영문명은 YONSEI, 슬로건은 Leading the Way to the Future이다.
이를 출력하는 프로그램을 작성해보도록 하자.

입력 :

첫째 줄에 N이 주어진다. (N = 0 또는 1)

출력 :

N = 0일 경우: 연세대학교의 영문명을 출력한다.
N = 1일 경우: 연세대학교의 슬로건을 출력한다.
대소문자 구별에 주의하도록 하자.

간단한 if문과 출력 방식을 안다면 풀 수 있는 문제입니다.
N == 1이라면 Leading the Way to the Future 를 출력하고, 아니라면 YONSEI 를 출력하면 됩니다.

+) 해보시면 아시겠지만 사실 if 문 자리에 N == 1 대신 N 만 써도 통과가 됩니다.
N이 양수라면 True로 취급하고, N이 0이면 False 취급하기 때문입니다.
파이썬에서는 N이 음수여도 True 취급하지만, 언어별로 음수는 False 취급하는 경우, 아예 if 자리에 정수만 넣을 수 없는 경우 등등이 있을 수 있기에 주의하셔야 합니다.
