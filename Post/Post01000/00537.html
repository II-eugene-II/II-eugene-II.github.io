문제 :

-

출력 :

-

$n$개의 정식 라운드와 $1$개의 테스트 라운드 이므로 총 라운드 수는 $n + 1$입니다.
각각의 라운드에 $2$개의 문제나 $3$개의 문제가 나오므로, 최솟값은 $(n + 1) \times 2$, 최댓값은 $(n + 1) \times 3$ 입니다.
입력은 n = int(input())으로 받아줍니다. 

        n = int(input())
        print((n + 1) * 2, (n + 1) * 3)
애초에 입력을 n = int(input()) + 1 로 받으면 더 편할 수 있습니다. 
        n = int(input()) + 1
        print(n * 2, n * 3)
