코탄젠트함수

코탄젠트함수 $\cot x$는 다음과 같이 정의합니다.

$$ \cot x = \frac{1}{\tan x} = \frac{\cos x}{\sin x} $$
$\tan x$는 탄젠트함수 , $\cos x$는 코사인함수 , $\sin x$는 사인함수 입니다.
즉, 탄젠트함수의 역수입니다.
탄젠트함수 자체는 정의가 안되는 부분에서 코탄젠트 함수가 정의 되는 부분이 있으므로 코사인함수와 사인함수의 비로 나타내는게 더 정확할 수도 있습니다.
코탄젠트함수의 멱급수 표현에 베르누이 수 가 사용됩니다.
코탄젠트함수의 역함수로 아크코탄젠트함수 가 있습니다.
코탄젠트함수의 미분 $$ \frac{{\rm d}}{{\rm d}x} \cot x = -\csc^2 x $$
코탄젠트함수 미분 증명

기본 전제조건 - $\frac{{\rm d}}{{\rm d}x} \sin x = \cos x$, $\frac{{\rm d}}{{\rm d}x} \cos x = - \sin x$ 임을 알고 있어야 함.
몫의 미분법 이용
$$ \frac{{\rm d}}{{\rm d}x} \cot x = \frac{{\rm d}}{{\rm d}x} \frac{\cos{x}}{\sin{x}} = \frac{-\sin x \times \sin x - cos x \times \cos x}{\sin^2 x} = -\frac{\sin^2 x + \cos^2 x}{\sin^2 x} $$ 삼각함수 항등식 $\sin^2 x + \cos^2 x = 1$ 에서 $-\frac{\sin^2 x + \cos^2 x}{\sin^2 x} = -\frac{1}{\sin^2 x}$이므로, $$ \frac{{\rm d}}{{\rm d}x} \cot x = -\frac{\sin^2 x + \cos^2 x}{\sin^2 x} = -\frac{1}{\sin^2 x} = -\csc^2 x $$ 이다.

삼각함수 항등식 $1 + \cot^2 x = \csc^2 x$으로 다음과 같이 쓸 수 있습니다.
코탄젠트함수의 미분 $$ \frac{{\rm d}}{{\rm d}x} \cot x = -(1 + \cot^2 x) $$
