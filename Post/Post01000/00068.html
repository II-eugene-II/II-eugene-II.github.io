문제 :

홍익대학교는 1946년에 개교하였다.
특정 년도가 주어졌을 때, 그 해가 개교 몇 주년인지 출력하라.
단, 홍익대학교는 없어지지 않는다고 가정한다.
문제는 C, C++, JAVA 또는 Python3을 이용하여 해결한다.
(...이하 생략...)

입력 :

입력으로 첫 줄에 특정 년도를 알리는 정수 N이 주어진다. 정수 N은 1,946 부터 1,000,000 사이의 값이다. (1,946 ≤ N ≤ 1,000,000)

출력 :

출력으로 홍익대학교의 개교 주년을 나타내는 정수를 출력한다.


1946년 개교, 1947년 개교 1주년, 1948년 개교 2주년이므로, $N$년에는 개교 $N-1946$주년 입니다.
정수를 입력 받아서, 1946을 뺀 후에 다시 출력해줍니다.
파이썬으로 정수를 입력 받으려면 int() 함수 안에 input() 을 넣어서 int(input()) 으로 써줍니다.
input()으로 1946을 입력받으면 "1946"이라는 문자열 상태이기 때문에, int() 함수로 감싸주어서 정수형 변수로 변환시켜주는 과정입니다.

한꺼번에 처리해줄 수도 있습니다.
