나눗셈 정리

나눗셈 정리 (Division Theorem) 은 다음과 같습니다.

임의의 두 양의 정수 $a$, $b$에 대하여 $$ a = bq + r $$ $(0 \leq r < b)$인 정수 $q$, $r$이 유일하게 존재한다.
평범해보이는 정리이지만 우선 이런 정리부터 깔고 가야 다른 정리들을 깔끔하게 정리할 수 있습니다.
나눗셈 정리 증명

(증명의 스케치)
$a + b > 0$임은 쉽게 알 수 있다. (전제가 $a$, $b$ 모두 양수였으므로)
따라서 다음과 같은 집합 $S = \{ a - nb \mid n \in \mathbb{Z}, \, a - nb \geq 0 \}$은 공집합이 아니다. ($n = -1$일 때 원소가 있음)

$S$에는 최소 원소가 존재함을 보이기

$S = \{ a - nb \mid n \in \mathbb{Z}, \, a - nb \geq 0 \}$ 에서 만약 $0 \not\in S$이면 $S = \{ a - nb \mid n \in \mathbb{Z}, \, a - nb \geq 0 \} = \{ a - nb \mid n \in \mathbb{Z}, \, a - nb > 0 \}$이므로, $S$가 자연수 집합 $\mathbb{N}$의 공집합이 아닌 부분집합이므로 자연수의 정렬 원리 에 따라 최소 원소가 존재한다.
$0 \in S$ 인 경우에는 집합 $S$에 $0$보다 작은 원소가 있을 수 없으므로 $S$의 최소 원소는 $0$으로 존재한다.
(자연수의 정렬 원리에서 자연수에 $0$이 포함된 상태로 증명한 경우에는 그냥 $S$가 자연수 집합 $\mathbb{N}$의 공집합이 아닌 부분집합이므로 자연수의 정렬 원리에 의해 최소 원소를 가진다고 할 수 있다.)

따라서 집합 $S$에는 최소 원소가 존재하고, 그 최솟값을 $r$이라 하면 어떤 정수 $q$에 대해 $a - bq = r$이라 할 수 있다.
$r$의 크기를 따져야 하는데, $r \geq b$이면 $a - b(q + 1) = r - b \leq 0$에서 $r - b$도 $S$의 원소가 되어서 $r$이 최솟값이라는데에 모순이므로 $r < b$이다.
유일함을 보이는 과정은

베주 항등식 등을 증명하는데에 사용됩니다.
