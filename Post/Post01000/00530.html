백준 티어 Platinum II 달성

백준 티어 플레 2 달성을 했습니다.
굳이 티어에 연연하고 뭐 일부러 점수를 올리고 그러고 싶지는 않았지만 Solved.ac의 시즌이 바뀌면서 기왕이면 시즌"2" 플레티넘"2" 배경이 예쁘지 않을까 싶어서 급하게 플레2에 도달했습니다.
세그먼트 트리라는 자료구조로 단 3일만에 20점 가까이 급하게 올려서 억지로 플레 2를 만들어서 솔직히 실감은 잘 나지 않습니다. 한동안은 플레 2로 계속 유지가 되지 않을까 싶습니다.
앞으로 남은 티어는 12개 (플레에 남은거 1개, 다이아 5개, 루비 5개?, 마스터??) 가 있는데, 점수 1점 올리는 일이 굉장히 어려워집니다. 세그먼트 트리는 굉장히 특수한 경우입니다.
플레 3 달성 달성 때는 다이아 어디선가 멈출거 같다고 했는데 아마 플레 2가 끝이지 않나 싶습니다. (곧 군입대이기도 하고...)
플레 3 때 750문제 푼 상태, 플레 4 때 1700문제 푼 상태, 플레 3은 2800문제를 넘게 푼 상태에서 달성했으니 아마 4000문제는 풀어야 티어가 바뀔 듯 합니다.
아직 부족하다고 생각되는 점 / 느낀 점을 하나 둘 적어보면서 앞으로의 방향을 잡아보자면...

1. 균형잡힌 컴공과같은 듯

자료구조가 꽤나 많이 오른게 맘에 듭니다. 기하학도 오르긴 했는데 사실상 미적분학에 가깝습니다.
스택 관련한 이것저것도 풀고 세그먼트 트리도 공부하고, 각종 그래프 탐색도 간단히는 할 수 있게 되었습니다.
이제 큐, 덱 이런 것들을 (뭔지는 알지만) 구현도 해보고 해야겠습니다.

2. Class 4

개인적으로 진짜 실력은 Class 라고 생각하는 편인데, 클래스 4까지는 임박했고, 3문제만 더 풀면 Class 4를 얻게 됩니다.
Class 2도 완전히 끝내는 것 (소위 금장) 까지 딱 두 문제 남았습니다.
문제를 아주 뻘로 풀지는 않았구나 싶습니다.

3. 파이썬이 너무 편하..긴 한데 이제 좀 느리다

파이썬이 참 이것저것 구현하긴 좋은데 이제 어려운 문제를 풀다보니 상당히 느리다는걸 느끼게 됩니다.
그렇다고 파이썬 말고 C++ 같은 걸 써보기에는 이미 파이썬에 절여진 바람에 정수 오버플로우 자체를 인지하지 못하기도 합니다. (1억 짜리 100만번 더해놓고 왜 값이 제대로 안뜨지? 같은...)
아마 정말 C++ 을 건들기 시작하면 다른 모든 것을 제치고 BigINT와 그에 따른 각종 템플릿을 만들어버리고 시작할 듯 합니다.

플레5 시점에서는 풀었던 문제 티어 Top100에서 가장 높은게 다이아4, 낮은게 실버3(!)이었고,
플레4 시점에서는 풀었던 문제 티어 Top100에서 가장 높은게 다이아4, 낮은게 골드4였습니다.
플레3 시점에서는 풀었던 문제 티어 Top100에서 가장 높은게 다이아2, 낮은게 골드3이었습니다.
이제는 가장 높은게 다이아2, 가장 낮은게 골드1이 되었습니다.
사실상 세그먼트 트리 기본 티어가 골드1이어서 생긴 일이기도 합니다.
가장 티어가 높은 100문제의 상태가 플레 3 시점에서 푼 문제는 이랬었는데, 플레 2 시점에서는 얼마나 달라졌는지 확인해보겠습니다.
